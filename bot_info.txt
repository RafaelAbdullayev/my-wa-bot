WhatsApp AI Moderator Bot - DeepSeek Pro 
================================= 
 
Python Server: http://127.0.0.1:8000 
 
Команды: 
- Очистить предупреждения: curl -X POST http://127.0.0.1:8000/clear_all_warnings 
- Проверить статус: curl http://127.0.0.1:8000/health 
- Добавить казино: curl -X POST http://127.0.0.1:8000/add_to_blacklist/example.com 
 
Файлы: 
- warnings.json - предупреждения пользователей 
- vt_cache.json - кэш VirusTotal 
- .env - API ключи
